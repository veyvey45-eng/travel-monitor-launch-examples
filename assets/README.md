# Visibility assets

Ce dossier contient désormais les **actifs visuels réutilisables** destinés à la diffusion externe du dépôt d’exemples et de la page Apify associée.

| Fichier | Rôle recommandé | Usage principal |
|---|---|---|
| `annotated_feed_item.svg` | Mettre en avant un **signal d’opportunité tarifaire concret** | Post LinkedIn, post X, page d’exemples, outreach léger |
| `annotated_first_success_flow.svg` | Montrer le **parcours de premier succès en moins d’une minute** | README, documentation d’onboarding, tutoriel, communauté |

Ces visuels sont générés à partir du **payload démonstratif réel** déjà présent dans `examples/demo_feed_item.json`. Ils servent de base propre pour la prochaine vague de diffusion externe, avant éventuelle déclinaison en captures raster ou carrousels plus travaillés.
